# Morona Lodge Booking System

This repository contains the booking & invoice system for Morona View Lodge.

## Features

- Booking calendar with unlimited year support
- Room availability
- Custom invoices matching the lodge layout
- Deposit & payment status
- WhatsApp invoice sending
- Excel export
- Invoice locking after checkout

## Installation

1. Upload the `bookings` folder to your server: `public_html/bookings`
2. Import `database.sql` in phpMyAdmin
3. Update database settings in `includes/db.php`
4. Visit `/bookings` in your browser to login
